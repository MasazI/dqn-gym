# dqn-gym
a practice imprimentation of dqn with open-ai gym.

# requirements
- python 3.6+
- tensorflow 1.0+
- keras 2.0+
- scipy
- numpy

# how to use
'''
python dqn.py --mode=<train or test or random>
'''

- train: train Deep Q-Network
- test: test using trained parameters
- random: playing random
